# Qual é o seu Pokémon?

Aplicação feita com HTML, CSS e JavaScript para sortear um Pokémon aleatório.

## O que o projeto faz

* Sorteia um Pokémon
* Mostra nome, imagem, altura e peso
* Funciona no celular
* Pode ser instalado como aplicativo
* Vibra no celular quando sorteia

## Tecnologias usadas

* HTML
* CSS
* JavaScript
* PokeAPI

## Arquivos do projeto

```text
index.html
style.css
script.js
manifest.json
service-worker.js
```

## Como usar

1. Abra o projeto
2. Clique no botão “Sortear Pokémon”
3. Veja qual Pokémon apareceu

## API usada

[https://pokeapi.co/](https://pokeapi.co/)
